== Description ==
 
Make your twitter feed appear on your wordpress blog. <br>
Note: The client must have internet access for this plugin to work - does not require server to have internet access.

== Installation ==

<h3>Install</h3>

1. Activate Plugin.<br>

2. Go to Settings>> In Twitter<br>

3. Enter your Username(you dont need the @ symbol), your twitter id, height(standard height is 211), how many twitter feeds you want displayed, and  the border colour you would like.<br>
You need to get your twitter ID go to <a hreaf="http://www.iannash.com/intwitter/where-to-find-your-twitter-widget-id/">http://www.iannash.com/intwitter/where-to-find-your-twitter-widget-id/</a> for instructions<br><br>

4.  Press "Update" <br>

5. Go to Appearance >> Widgets >> and place the widget into the sidebar.

<h3>To Uninstall</h3>

1. Remove the textbox widget
2. Deactivae the plugin
